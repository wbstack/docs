# Queryservice

The queryservice is made up of a number of components:

  • queryservice (opens new window) - Blazegraph SPARQL query service (WDQS flavour)
  • queryservice-gateway (opens new window) - Custom gateway proxy to route traffic to the correct Blazegraph namespace based on request domain.
  • queryservice-updater (opens new window) - Custom updater to updater multiple sites from a single process.
  • queryservice-ui (opens new window) - Fork of the WDQS UI with modifications for wbstack.
